# RankHub - 音游数据管理平台

RankHub 是一个基于 Flutter 开发的跨平台音游数据管理应用，支持多平台账号管理、数据同步、成绩查询和其他功能。

幻想打造一个聚合音游平台。

## 重要通知

RankHub 项目归属权变更公告

各位群成员、贡献者与用户：
Project Fukakai 已与 汇智卓创 达成协议并完成项目收购，按行业惯例，本次收购未达公示普遍底限金额，具体金额将不会公开。

2. 贡献补偿：特设专项激励基金500元，将对除主、次开发者外的所有贡献者，按 2026年3月22日00:00 GitHub提交统计的贡献比例和开发者提交的资料，统一分配补偿奖励。
3. 申领方式：请贡献者（贡献形式不限研发/宣发/社区管理）于即日起，在14日内，通过发送邮件到 qianmo@smart-teach.cn 提交联系方式与贡献证明（材料形式不限，GitHub截图或自行举证均可），逾期视为自动放弃。
4. 后续说明：闭源版本将独立迭代，原有开源版本按原协议保留历史授权，不再更新维护。
5. 周期提示：自本公告发出起，7日内可提出异议，公示将保留置顶至少14日，在此期间，仓库代码仍可公开访问，20日内全部款项将结清。

几个值得注意的点：
